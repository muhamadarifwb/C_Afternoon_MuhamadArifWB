ABOUT ME

Muhamad Arif Wibowo

Saya Seorang mahasiswa yang bersemangat dari Universitas Bina Sarana Informatika, saat ini telah mencapai semester 7 dalam jurusan Teknologi Informasi. Dalam perjalanan akademik, saya telah menunjukkan minat dan komitmen yang kuat terhadap dunia pengembangan Frontend. Dengan pengalaman belajar lebih dari 4 tahun di dunia Teknologi Informasi, Saya telah mengasah keterampilan yang sangat dibutuhkan dalam pengembangan Frontend.

Tiga komponen utama dalam pengembangan web, yaitu HTML, CSS, dan JavaScript, menjadi teman sehari-hari bagi seorang frontend Development. HTML digunakan untuk membangun struktur dasar sebuah halaman web, CSS digunakan untuk mengatur tampilan dan tata letak, dan JavaScript untuk memberikan interaktivitas pada situs web. Saya telah berhasil menguasai HTML untuk membuat struktur halaman yang kuat dan efisien. Dengan kemampuan saya dalam menggunakan CSS, saya dapat mengubah tampilan halaman web dengan sentuhan artistik yang indah. Selain itu, saya juga memiliki pemahaman dasar tentang JavaScript yang memungkinkannya untuk menambahkan fungsi dan interaksi pada halaman webnya.

Selama masa studi, saya telah mengambil berbagai kursus terkait Frontend Development, seperti Desain Responsif dan Pengembangan Web Interaktif. Selain itu, saya juga aktif dalam proyek-proyek kuliah yang melibatkan pengembangan aplikasi web Frontend. Selain keterampilan teknis, saya juga memiliki kemampuan berpikir kreatif dan desain yang kuat. Saya selalu berusaha untuk menciptakan pengalaman pengguna yang menarik dan intuitif melalui desain tampilan yang efektif.

Dengan semangat yang tak kenal lelah, Sebagai mahasiswa Universitas Bina Sarana Informatika ini berharap untuk terus mengembangkan keterampilan Frontend Development. Saya melihat masa depan yang cerah di dunia teknologi informasi dan siap untuk menghadapinya dengan penuh semangat serta kompetensi yang dimiliki di bidang Frontend Development.
